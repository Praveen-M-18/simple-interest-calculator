# Contributing to Simple Interest Calculator

Thank you for taking the time to contribute! All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

### 1. Reporting Bugs
* Check the existing Issues tab to make sure the bug hasn't already been reported.
* If it is a new bug, open a new Issue and describe the problem, including steps to reproduce it and your expected vs. actual outcomes.

### 2. Suggesting Enhancements and Ideas
* If you have an idea or enhancement to make the calculator better, feel free to open an Issue to discuss it first.
* Explain the utility of the feature and how it benefits the project.

### 3. Code Fixes & Pull Requests
* Fork the repository and create a new feature branch for your changes.
* Submit a Pull Request (PR) with a clear description of the bug fixes or documentation improvements you've made.
