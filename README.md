# simple-interest-calculator

A simple bash/python script or application that calculates the simple interest based on the principal amount, annual interest rate, and time period.

## Formula
The simple interest is calculated using the standard formula:
A = P(1 + rt)

Where:
* **P** = Principal amount
* **r** = Annual interest rate (in decimal format)
* **t** = Time period (in years)

## Usage / Example
1. Input Principal: 1000
2. Input Rate of Interest (%): 5
3. Input Time (years): 2

**Output:**
* Total Interest: 100
* Total Amount: 1100
